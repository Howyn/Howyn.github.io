有效的括号

我的初次题解

解题思路：

  • 用到栈的思想。是要找匹配，如果出现任意一种左边括号，比如{，如果下一个括号是右括号，必须是}，否则直接false。
  • 所以左括号直接入栈，右括号，如果匹配就把左括号出栈，不匹配，就直接false。
  • 在循环外也要判断，因为如果一直是左括号，或者出现这种([{{ } ，也会直接退出循环。所以退出循环后，判断栈是否为空，为空则代表true，否则false。
  • 首先进行奇偶判断，可以加快效率，如果是奇数，就不用进入循环了。

代码知识点

C# Stack 栈

前置：必须引入命名空间

  1. 栈的创建（泛型 Stack）

知识点：Stack<char> 是存储字符的后进先出容器，<char> 指定只能存字符类型

  1. Push () 入栈（本题核心：存左括号）

知识点：将元素放到栈最顶部，无返回值；方法首字母大写 Push，小写 push 会报错

  1. Peek () 查看栈顶元素（不删除）

知识点：读取当前栈最上面的字符，不会移除元素；栈为空时调用会直接抛异常，所以必须先判断 stack.Count > 0

  1. Pop () 弹出栈顶元素（取出 + 删除）

知识点：取出栈顶字符并从栈中移除；空栈调用报错，必须提前判 Count

  1. Count 属性（判断栈是否为空）

知识点：stack.Count 获取栈内现有元素数量，只读属性

LIFO 后进先出：最后存入的左括号，必须最先匹配闭合，完美适配括号嵌套规则。

c#字符串知识点

1.string.Length 获取字符串总长度

知识点：string 是只读字符数组，.Length 只读属性，返回字符总数，用于奇偶剪枝

2.下标访问单个字符 s [i]

知识点：字符串可以像数组一样通过数字下标取对应位置 char，下标从 0 开始；返回类型是 char（单引号字符）

3.for 循环遍历字符串全部字符

知识点：配合 Length、下标 i 逐个取字符，本题使用的遍历方式

  1. char 字符等值判断（极易踩坑点）

知识点：

  • char 类型：单引号 ‘(‘
  • string 类型：双引号 “(“
